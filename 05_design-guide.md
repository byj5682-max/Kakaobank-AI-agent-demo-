# 카카오뱅크 AI 가계부 에이전트 — 디자인 가이드

**버전:** v1.0 | **작성일:** 2026-05-07

> 카카오뱅크 기존 디자인 언어와 충돌 없이 align하되,
> AI 기능 전용 요소는 "한 단계 따뜻하고 능동적인" 톤으로 구분한다.

## 2. 타이포그래피

> 카카오뱅크 앱과 동일하게 **Apple SD Gothic Neo** (iOS) / **Noto Sans KR** (Android) 사용

### 2.1 타입 스케일

| 레벨 | 크기 | 굵기 | 행간 | 용도 |
|------|------|------|------|------|
| **Display** | 32px | 800 | 1.2 | 금액 강조 (예상 지원금, 절약 금액) |
| **Title L** | 22px | 700 | 1.3 | 화면 제목 |
| **Title M** | 17px | 700 | 1.4 | 네비바 제목, 카드 제목 |
| **Title S** | 15px | 700 | 1.4 | 섹션 제목, 버튼 텍스트 |
| **Body M** | 14px | 400 | 1.55 | 본문, 설명 텍스트 |
| **Body S** | 13px | 400 | 1.55 | AI 코멘트, 부가 설명 |
| **Caption** | 11px | 400 | 1.4 | 날짜, 레이블, 단위 |

### 2.2 핵심 원칙

- 금액은 항상 **굵게 (700 이상)**, 단위(원)는 한 단계 작게
- AI 카피는 **Body S (13px)**, 줄간격 넉넉하게 (1.6)
- 긴박 상태 텍스트는 크기 유지, **컬러로만** 위급함 표현 (크기 키우지 않음)

## 3. 그리드 & 스페이싱

### 3.1 기준 그리드

- **기준 단위:** 4px
- **화면 좌우 여백:** 16px (콘텐츠 영역 343px)
- **카드 내부 패딩:** 16px
- **컴포넌트 간격:** 10~12px
- **섹션 간격:** 20~24px

### 3.2 스페이싱 토큰

| 토큰 | 값 | 용도 |
|------|-----|------|
| `space-xs` | 4px | 아이콘–텍스트 간격 |
| `space-sm` | 8px | 인라인 요소 간격 |
| `space-md` | 12px | 컴포넌트 내부 행 간격 |
| `space-lg` | 16px | 카드 패딩, 화면 여백 |
| `space-xl` | 24px | 섹션 간격 |

## 4. 컴포넌트 가이드

### 4.1 홈 AI 카드

```
┌─────────────────────────────────┐  ← 배경: #FFFBE6, 테두리: #F5D700 점선
│  🌟 AI 가계부          [태그]   │  ← Caption, Gray 500
│                                 │
│  지영님, 우리집 가계부          │  ← Title S, Gray 900
│  정리됐어요.                    │
│                                 │
│  지출 흐름 확인하시고,          │  ← Body S, Gray 600
│  필요한 할일 챙겨보세요.        │
│                                 │
│                    확인하기 →   │  ← Body S Bold, Gray 900, 우측 정렬
└─────────────────────────────────┘
```

- border-radius: 16px
- 탭 시 배경 `#FFF8CC`로 피드백

### 4.2 발견된 할일 카드 (Action Card)

```
┌─────────────────────────────────┐  ← 배경: White, 테두리: #E0E0E0
│  💰  고유가 피해지원금 신청하기  ›  │  ← 아이콘 24px + Title S + 화살표
│      우리 가족 최대 60만원        │  ← Caption, Gray 500
└─────────────────────────────────┘
```

- border-radius: 12px
- 탭 피드백: 배경 `#F5F5F5`

### 4.3 금액 강조 박스

```
┌─────────────────────────────────┐  ← 배경: #F7F7F7
│     우리 가족 예상 지원금        │  ← Caption, Gray 500, 가운데
│                                 │
│          60만원                 │  ← Display (32px, 800), Gray 900
│                                 │
│   1인당 15만원 × 4명            │  ← Caption, Gray 400, 가운데
└─────────────────────────────────┘
```

- border-radius: 14px
- 금액 위아래 padding: 20px 이상

### 4.4 자격 체크 리스트

- ✅ 아이콘: 22px
- 항목 간 구분선: `#F2F2F2` 1px
- 등장 애니메이션: 300ms 간격으로 순차 페이드인 + 좌→우 슬라이드

### 4.5 챌린지 진행 헤더

- 3개 수치(사용/목표/잔액) 가로 균등 배치
- 프로그레스 바 높이: 10px, 배경 `#EEEEEE`, fill `#191919`
- 잔액 수치: 정상 `#111111` → 주의 `#FB8C00` → 위험 `#E53935`

### 4.6 타임라인

- 왼쪽 세로선: `#E0E0E0` 2px
- 점(dot): 기본 `#CCCCCC` → 주의 `#FB8C00` → 위험 `#E53935`
- 날짜: Caption, Gray 400
- 한 줄 텍스트 + 우측 잔액: 한 행에 완결
- 행 간격: 14px

### 4.7 버튼

| 타입 | 배경 | 텍스트 | 용도 |
|------|------|--------|------|
| Primary | `#FEE500` | `#191919` | 주요 CTA (신청 알림, 참여하기) |
| Secondary | `#FFFFFF` | `#444444` | 보조 액션 (홈으로 돌아가기) |

- 높이: 52px
- border-radius: 12px
- 텍스트: Title S (15px, 700)
- Primary 버튼은 화면당 1개만

## 5. AI 전용 디자인 원칙

### 5.1 AI 카드와 일반 카드 구분

| 구분 | AI 카드 | 일반 카드 |
|------|---------|----------|
| 배경 | `#FFFBE6` (Soft Yellow) | `#FFFFFF` |
| 테두리 | `#F5D700` 점선 | `#E0E0E0` 실선 |
| 아이콘 | 🌟 고정 | 기능별 다름 |

### 5.2 AI 카피 원칙

| 원칙 | 나쁜 예 | 좋은 예 |
|------|---------|---------|
| **짧게** | "4월 한 달 동안의 외식비 지출 내역을 분석한 결과..." | "4월 외식비가 30만원 더 나왔어요." |
| **행동 유도** | "외식비가 증가했습니다." | "아래 할일 챙겨보세요." |
| **사용자 편** | "조건이 충족되었습니다." | "조건 세 가지 모두 충족이에요! 🎉" |
| **이름 호칭** | "고객님" | "지영님" |
| **긴박할 때** | 크기 키우기 | 컬러만 `#E53935`로 변경 |

### 5.3 이모지 사용 규칙

| 상황 | 이모지 |
|------|--------|
| AI 카드 태그 | 🌟 |
| 지원금 | 💰 |
| 외식/음식 | 🍽️ |
| 챌린지 성공 | 🎉 |
| 저금통 | 🐷 |
| 주의 경고 | ⚠️ |
| 위험 경고 | 🚨 |
| 응원 | 💪 |
| 자녀장려금 예고 | 💡 |
| 알림 | 🔔 |

- 이모지는 문장 **앞**에만 사용, 뒤에 붙이지 않음
- 한 문장에 이모지 1개 원칙

## 6. 화면 전환 & 모션

| 전환 | 방식 | 시간 |
|------|------|------|
| 홈 → AI 가계부 | 아래에서 슬라이드업 (sheet) | 280ms |
| 가계부 → 지원금/챌린지 | 오른쪽에서 슬라이드인 | 250ms |
| 자격 체크 등장 | 순차 페이드인 + translateX(-12px→0) | 400ms, 350ms 간격 |
| 버튼 탭 | 배경색 즉시 변경 | 즉시 |
| 토스트 | 페이드인/아웃 | 300ms |

- easing: `ease` 기본
- 과도한 모션 금지 — 정보 전달이 우선
