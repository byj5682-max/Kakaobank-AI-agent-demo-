# 카카오뱅크 AI 가계부 에이전트 — 요구사항 정의서

**프로젝트명:** AI 가계부 에이전트 (가칭: "카뱅 AI 총무")
**버전:** v1.0
**작성일:** 2026-05-06
**작성자:** 릴리 (기획)

## 1. 프로젝트 개요

### 1.1 목적

카카오뱅크 사용자(특히 40대 맞벌이 가구)가 가계부 확인에서 정부 지원금 신청, 지출 챌린지 참여까지 **AI가 한 흐름으로 연결**하여 행동을 완료할 수 있도록 돕는다.

> **핵심 thesis:** 금융 AI는 "정보를 잘 보여주는 것"에서 "사용자 행동을 먼저 돕는 것"으로 진화해야 한다.
> 사용자가 진짜 원하는 건 정보 자체가 아니라 **"그래서 지금 나는 어떻게 해야 하지?"** 다.

### 1.2 배경

| 항목 | 현황 |
|------|------|
| 카카오뱅크 고객 수 | 2,727만 명 (2026년 1Q) |
| MAU | 2,032만 명 |
| 40대 침투율 | **80%** (핵심 타겟층) |
| 신규 가입자 중 우리아이통장 | **24%** (자녀 가구 핵심 성장 동력) |
| 대화형 AI MAU 성장 | 4개월 만에 **10배** |
| 기존 정부지원금 찾기 가입자 | **100만 명** 돌파 |

### 1.3 진화 포인트

```
현재 카뱅                          우리가 제안하는 단계
─────────────────────              ──────────────────────────────
AI 인사이트 카드 (홈 노출)   →     인사이트 + 액션 카드 (홈 노출)
정부지원금 메뉴 (사용자가 찾기) →  자격 자동 판별 + 신청까지 한 번에
챗봇 Q&A (수동 질문)        →     선제적 AI 추천 (전용 화면)
```

## 2. 서비스 범위

### 2.1 In Scope (이번 Demo 포함)

| # | 기능 | 설명 |
|---|------|------|
| F-01 | **홈 AI 카드** | 카뱅 홈 상단에 개인화된 AI 액션 카드 노출 |
| F-02 | **AI 가계부 합산 뷰** | 본인 + 배우자 + 모임통장 수입/지출 합산 시각화 |
| F-03 | **고유가 피해지원금 자격 자동 확인** | 소득·가구원수·거주지 기반 자동 판별 |
| F-04 | **고유가 피해지원금 신청 연결** | 카뱅 내 신청 메뉴(2026.04.27 신설)로 원클릭 이동 |
| F-05 | **외식비 챌린지 설정** | 전월 지출 분석 → 목표 한도 설정 → 참여 |
| F-06 | **챌린지 진행 트래킹** | 카드·이체·모임통장 지출 자동 감지 + 잔액 알림 |
| F-07 | **챌린지 성공 보상** | 축하금 + 차등 α → 카뱅 저금통 자동 입금 |
| F-08 | **자녀장려금 예고 안내** | 하단 한 줄 배너 (5월 신청 시즌 예고) |

### 2.2 Out of Scope (Demo 제외, 향후 확장)

- 자녀장려금 자격 자동 판별 및 신청 (구조는 동일, Demo에서는 예고만)
- 보험·대출 상품 추천
- 외부 서비스 구독 관리 (실제 해지 불가)
- 순수 챗 Q&A 형식 UI (홈 카드 탭 시 새 챗창으로 진입하되, 화면 내 구성은 카드·전용 뷰·리스트 등 다양한 형식으로 자유롭게 구현 — 챗창은 진입 방식일 뿐, 내용은 챗 형식이 아님)

## 3. 기능 요구사항

### F-01. 홈 AI 카드

| 항목 | 내용 |
|------|------|
| 노출 위치 | 카카오뱅크 홈 화면 상단 (기존 AI 인사이트 카드 영역) |
| 트리거 조건 | 월 초(1~7일) 또는 주요 이벤트 발생 시 (지원금 신청 시즌 등) |
| 카드 메시지 | "🌟 지영님, 우리집 가계부 정리됐어요. 지출 흐름 확인하시고, 필요한 할일 챙겨보세요." |
| CTA | [탭하여 보기] → AI 가계부 화면 이동 |
| 우선순위 | Must Have |

### F-02. AI 가계부 합산 뷰

| 항목 | 내용 |
|------|------|
| 데이터 범위 | 본인 계좌 + 배우자 연결 계좌 + 모임통장 |
| 표시 항목 | 이번 달 수입 합계 / 지출 합계 / 카테고리별 지출 |
| 하이라이트 | 전월 대비 급증 카테고리 강조 (예: 외식비 +30만원 ↑) |
| 액션 카드 | 가계부 하단에 발견된 액션 2개 카드 노출 |
| 우선순위 | Must Have |

### F-03. 자격 자동 확인 (고유가 피해지원금)

| 항목 | 내용 |
|------|------|
| 판별 기준 | ① 소득 하위 70% 여부 ② 가구원 수 (4인) ③ 거주지 (비수도권) |
| 데이터 소스 | 카뱅 보유 금융 데이터 + 기가입 정보 활용 |
| 결과 표시 | 각 조건별 ✅ / ❌ 체크 + 예상 수령액 표시 (예: 4인 가구 비수도권 기준 60만원) |
| 우선순위 | Must Have |

### F-04. 지원금 신청 연결

| 항목 | 내용 |
|------|------|
| 연결 대상 | 카뱅 내 고유가 피해지원금 신청 메뉴 (2026.04.27 신설, 공지 18760) |
| 신청 일정 | 소득하위 70%: **2026.5.18(월) ~ 7.3(금)** (2차) |
| 지급 수단 | 지역사랑상품권 / 신용·체크카드 / 선불카드 중 선택 |
| 사용 기한 | 2026.8.31까지 (주소지 기준 소상공인 매장, 지역사랑상품권 가맹점) |
| 사전 알림 | 신청 전(5.1 시점)이면 "5월 18일 신청 시작, 알림 설정해드릴게요" 안내 |
| 사후 안내 | "신청 완료되면 지영님 카뱅 카드로 바로 쓸 수 있도록 안내해드릴게요" |
| 콜센터 | 1670-2626 (화면 내 표시) |
| 우선순위 | Must Have |

### F-05. 외식비 챌린지 설정

| 항목 | 내용 |

## 6. 성공 지표 (KPI)

### 전제: 현재 AI MAU 추정
카카오뱅크는 2025.11 AI 이체, 2025.12 AI 모임 총무 출시 후 2026.04 기준 대화형 AI MAU가 4개월 만에 10배 성장했다고 발표. 전체 MAU 2,032만 중 약 **600만 추정** (보수적 기준).

### 핵심 KPI: AI MAU 상승

| 지표 | 현재 (2026.04) | 목표 (2026.10, 도입 6개월 후) |
|------|--------------|------------------------------|
| **AI MAU** | ~600만 (추정) | **900만 이상** (+50%) |
| 챌린지 인지율 | — | AI MAU의 50% = 300만 |
| 챌린지 참여 | — | 인지의 30% = **90만/월** |
| 챌린지 성공 | — | 참여의 40% = **36만/월** |
| 알림 경유 앱 재방문 | — | 참여자 1인당 월 평균 4회↑ |

### 운영 비용 시뮬레이션 (성공자 기준)

| 시나리오 | 1인당 평균 보상 | 월 비용 | 연 비용 |
|---------|--------------|--------|--------|
| 보수 (기본 1,000원만) | 1,000원 | 3.6억 | **43억** |
| 중간 (tier mix 평균) | ~2,000원 | 7.2억 | **86억** |
| 공격 (전원 최대 3,000원) | 3,000원 | 10.8억 | **130억** |

> tier mix 가정: 0~3만 절약(40%) → 1,500원 / 3~6만(35%) → 2,000원 / 6만↑(25%) → 3,000원 → 평균 약 2,000원

### 비용 효율 비교

| 항목 | 금액 |
|------|------|
| AI MAU 1인당 연간 비용 (중간) | 86억 ÷ 600만 = **약 1,433원** |
| 신규 고객 CAC | 5,000~20,000원 |
| **비용 효율** | **CAC 대비 1/3~1/14** |

### 부가 가치 (정성 KPI)

| 항목 | 내용 |
|------|------|
| 저금통 락인 | 보상 저금통 입금 → 인출 마찰 → 수신 잔액 기여 |
| 가계부 재방문 | 알림 → 앱 재방문 → AI MAU 직접 기여 |
| 서비스 확장성 | 동일 구조로 자녀장려금·난방비 등 연중 확장 가능 |
